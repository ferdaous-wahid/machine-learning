  * Prediction
3. Evaluation
  * Calculate performance metrics
  * Make a verdict

**Feature Scaling is always applied to columns** <br>

##Types of feature scaling <br>

| `Normalization` | X' = (X-Xmin) / (Xmax-Xmin) | `[0 or 1]` | <br>
| `Standardization` | X' = (X - η) / σ | `[-3; +3]`
